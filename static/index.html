ABOUT
Video enhancement that actually runs on your phone.

What's Different

  • Runs on iOS natively via a-Shell. Not a cloud service—your phone is the GPU.
  • SIGINT crash recovery — interrupted encodes don't fail; they continue from where they left off.
  • Upscale-once, refine-many — scales once, refines up to 4 times with decaying strength for quality without bloat.
  • Portrait video native — auto-detects orientation and scales correctly for phone-shot vertical video.
  • No GPL/no libx264 — hardware VideoToolbox encoders only. No licensing headaches.
  • Self-calibrating estimates — learns your hardware speed over time, gets more accurate with each use.
  • Download + enhance in one step — paste a YouTube URL and get an enhanced video, or process entire channels.
  • Local or cloud GPU — same code, same presets. Works on your phone, your laptop, or rented compute.

How It Works

Download
→
Pass A: Cleanup
→
Pass B: Upscale + Refine
→
Enhanced MP4

Pass A deblocks and debands at source resolution. Pass B upscales once, then refines with decaying filter strength across 1-4 passes.

Presets

CLEAN

Deblock + deband only. Safe, minimal processing for already-decent video.

RESTORE

Adds DCT denoising for noise reduction. Good for compressed sources.

SHARP

Weak deblock + unsharp sharpening. For soft, blurry video.

CINEMATIC

Hue-saturation + vibrance + RGB curves. Color grading without artifact removal.

DEEP_CLEAN

Heavy deblock + large deband + FFT denoising. For heavily compressed sources. (slow)

STABILIZE

Deshake + deflicker. For shaky or flickering video.

iOS a-Shell Setup

1. Install a-Shell: Download the free a-Shell app from the App Store.

2. Install Python & FFmpeg:

pip install yt-dlp flask
a-install ffmpeg

3. Clone enxr:

git clone https://github.com/gleekr/enxr.git
cd enxr

4. Start the web UI:

python3 server.py

5. Open browser: Go to localhost:8080 in your iPhone browser.

Tip: use a keyboard shortcut in a-Shell to quick-start the server each time.
